关于本项目

这是一个展示全链路可观测性技术的个人项目，体现了从日志采集到可视化监控的完整流程。

技术栈

  • 前端: Flask + HTML/CSS/JS
  • 日志采集: Filebeat + Kafka
  • 数据处理: Python消费者 + ELK Stack
  • 监控告警: Celery + Redis + SMTP
  • 可视化: Kibana + Grafana
  • 系统监控: Prometheus
